## 🛠 기술 스택
- Front
    - Next.js
    - TypeScript
    - Router from Next.js
    - Redux Tool Kit

    - CSS Module

- Back
    - Java 17
    - Spring Boot , JPA
    - Spring Security, JWT
    - Gradle
    - MySQL
